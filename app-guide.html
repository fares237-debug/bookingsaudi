ملفات PWA المطلوبة (جاهزة للنسخ)
📄 manifest.json - أضفه في مجلد موقعك:
{ "name": "بوكنق السعودية", "short_name": "بوكنق", "description": "أكبر منصة حجوزات في السعودية", "start_url": "/index.html", "display": "standalone", "background_color": "#0A0A0F", "theme_color": "#C9A84C", "orientation": "portrait", "icons": [ {"src": "icon-192.png", "sizes": "192x192", "type": "image/png"}, {"src": "icon-512.png", "sizes": "512x512", "type": "image/png"} ] }
📄 service-worker.js - للعمل بدون إنترنت:
const CACHE = 'booking-saudi-v1'; const FILES = ['/', '/index.html', '/hotel-page.html', '/search-page.html', '/booking-page.html']; self.addEventListener('install', e => { e.waitUntil(caches.open(CACHE) .then(c => c.addAll(FILES))); }); self.addEventListener('fetch', e => { e.respondWith(caches.match(e.request) .then(r => r || fetch(e.request))); });
📄 أضف هذا في <head> لكل صفحة:
<link rel="manifest" href="/manifest.json"> <meta name="theme-color" content="#C9A84C"> <meta name="apple-mobile-web-app-capable" content="yes"> <meta name="apple-mobile-web-app-title" content="بوكنق السعودية"> <script> if('serviceWorker' in navigator) { navigator.serviceWorker.register('/service-worker.js'); } </script>
